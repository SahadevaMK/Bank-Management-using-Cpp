# Bank-Management-using-Cp

## This is mini project developed using basic concepts of cpp such as-
*Objected oriented programming(OOPS) 
*Loops
*Pointers
*Arrays
*Structures
*Conditional statements(if,else,elseif,switch)

# This project allows you to 
-Create account
-Delete account
-Depositing money
-Withdrawing money
-Quiting from bank
